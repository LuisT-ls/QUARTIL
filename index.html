Quartis - Análise de Distribuição

Os quartis são essenciais para entender a distribuição dos dados. Eles dividem os dados em quatro partes iguais, permitindo identificar outliers e analisar a dispersão dos valores.

Primeiro Quartil (Q1)

25% dos dados estão abaixo deste valor

Mediana (Q2)

50% dos dados estão abaixo deste valor

Terceiro Quartil (Q3)

75% dos dados estão abaixo deste valor

O que são Quartis e como calcular?

Os quartis são medidas estatísticas que dividem um conjunto de dados ordenados em quatro partes iguais, cada uma contendo 25% dos dados.

Primeiro Quartil (Q1)

Representa o valor que separa os primeiros 25% dos dados. É o 25º percentil.

Segundo Quartil (Q2)

É a mediana dos dados, separando os primeiros 50% dos últimos 50%.

Terceiro Quartil (Q3)

Representa o valor que separa os primeiros 75% dos dados. É o 75º percentil.
