# 🧮 Calculadora de Quartil e Estatística Online

Uma calculadora online gratuita e completa para análise estatística, com foco especial em **quartis (Q1, Q2, Q3)** e medidas estatísticas fundamentais.

## ✨ Características Principais

### 🎯 **Cálculo de Quartis**

- **Primeiro Quartil (Q1)**: 25º percentil
- **Segundo Quartil (Q2)**: Mediana (50º percentil)
- **Terceiro Quartil (Q3)**: 75º percentil
- **Amplitude Interquartil (IQR)**: Q3 - Q1
- **Detecção de Outliers**: Baseada na regra 1.5×IQR

### 📊 **Medidas Estatísticas Completas**

- **Medidas de Posição**: Média, Mediana, Moda
- **Medidas de Dispersão**: Desvio Padrão, Variância, Coeficiente de Variação
- **Análise Avançada**: Assimetria, Curtose

### 📈 **Visualizações Interativas**

- **Histograma**: Distribuição de frequência
- **Boxplot**: Visualização de quartis e outliers
- **Gráficos responsivos** com Chart.js

### 💾 **Exportação Multi-formato**

- PDF, CSV, XLSX, JSON, TXT
- Relatórios profissionais
- Dados estruturados para análise
